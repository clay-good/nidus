# Spec 10 — Population Modelling (superseded)

> **Status: Superseded by the v0.3 design.** This spec described cohort sampling and individual heterogeneity inside the Rust simulator. With the Rust prototype removed, there is nothing to sample. Population-level uncertainty is now expressed at the parameter level via `low` / `central` / `high` fields and distribution metadata in the JSON dataset.
